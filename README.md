## 👾 Sobre este proyecto

**Pixel Dev Portfolio** es mi sitio web personal diseñado con estética **Metroidvania / Pixel Art**, desarrollado completamente con **HTML5, Tailwind CSS y JavaScript vanilla**. 

El objetivo: mostrar mis servicios como desarrollador **Full Stack** especializado en **Videojuegos 2D** (Unity/Godot) y **Sistemas Backend** (Laravel/PHP) de una manera visual, interactiva y profesional.

> ✨ "Un portafolio que refleja mi pasión por los videojuegos retro y el código limpio."

---

## 🧙 Características principales

- 🎨 **Diseño Pixel Art / Metroidvania** con animaciones suaves
- 🧠 **Canvas animado** en el Hero con estética de juego 2D
- 📱 **Totalmente responsive** (Mobile-first)
- 🎴 **Portafolio interactivo** con efecto Flip Card al hacer hover
- 🎮 **Sección de servicios separada** (Web, Juegos, Apps) en Tabs
- 💰 **Planes y precios** claramente estructurados
- 📞 **Integración directa con WhatsApp** (sin formularios)
- ⚡ **Optimizado para SEO** y redes sociales (Open Graph)
- 🖼️ **Sección "Sobre mí"** con foto y datos de contacto
- 📊 **Barras de habilidades animadas** con niveles de dominio

## 🎨 Personalización

¿Quieres usar este diseño como base para tu propio portafolio?

1. **Cambia los colores** en las clases de Tailwind
2. **Reemplaza las imágenes** en `assets/images/`
3. **Actualiza los textos** y enlaces de WhatsApp
4. **Modifica los precios** en la sección de planes
